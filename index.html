The Light Bulb Problem

Switch Presses: 0

Congratulations! All bulbs are on!

Problem Statement

N light bulbs are connected by a wire. Each bulb has a switch associated with it. However, due to faulty wiring, a switch also changes the state of all the bulbs to the right of the current bulb. The challenge is to turn on all the bulbs with the minimum number of switch presses.

Greedy Algorithm Explanation

Greedy algorithms aim to make the optimal choice at that given moment. In this game, at each step, we select the leftmost bulb that's off and toggle its switch. This approach ensures that we're making a local optimal choice (turning the leftmost off-bulb on) with the hope of leading to a global optimal solution (turning all bulbs on in the minimum number of moves).
